# Tipografia
* estudo das técnicas de escritas
* glifos, letras, caracteres = signos alfabético projetados para reprodução mecânica
* família tipográfica = conjunto de glifos que possuem as mesma características anatômicas, independente de suas variações
* fontes = conjunto de glifos que formam uma família tipográfica
    * cuidar a legibilidade e a leiturabilidade
    * categoria de fontes:
        * fontes serifadas
        * fontes não serifadas
        * fontes monoespaçadas
        * fonte script
        * fonte display
* para configurar a familia tipografica que será aplicada --> propiedade font-family 
    * se for indica mais de uma familia o navegador dâ preferência para a primeira, se o navegador não tiver a fonte, ele vai para a segunda e assim sucessivamente. Caso não tenha nenhuma das fontes indicadas ela vai para a família genérica, geralmente, serif, sans-serif ou monospaced
* também é possível configurar o tamanho das fontes
    * medidas : centímetro(cm), polegadas(in), paicas(pc), pixels(px),...
        * medida em = medida referêncial em relação ao tamanho original da fonte, geralmente 16px
* outros estilos:
    * propiedade font-weight --> negrito, pode ser aplicado por lighter, bold, bolder ou por peso númerico
    * propiedade font-style --> itálico, normal ou oblique
* **shorthand** = atalho para formatação de fonte em css
    * shorthand font simplifica a configuração da fonte
    * ordem dos atributos de uma shorthand:
        * font-style
        * font-variant
        * font-weight
        * font-size/line-height
        * font-family
* alinhamento de texto 
    * text-align: left(esquerda) ou right(direita) ou center ou justify
