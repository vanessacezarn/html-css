# Curso-HTML5-e-CSS3
Curso de HTML 5 E CSS 3 do Curso em Vídeo do professor Gustavo Guanabara. (https://www.cursoemvideo.com/)

O curso é dividido em 5 módulos:
* Módulo 1:
  * conceitos básicos
  * preparação do ambiente
  * semântica do HTML 5
  * textos, títulos, ligações, multimidia  
* Módulo 2
  * fundamentos do design
  * psicologia das cores
  * tipologia
  * elementos CSS
  * modelos de caixa
  * Wire frame
  * responsividade
* Módulo 3
  * versionamento de software
  * hospedagem de sites estáticos
  * tabelas
* Módulo 4
  * quadro em linhas
  * formulários
  * media queries
  * mobile first
* Módulo 5
  * flex box
  * grid layout
  * projeto final
