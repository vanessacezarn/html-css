Semâtinca

Para deixar uma ou mais palavras em negrito é preciso usar a tag <strong> entre as palavras que se quer dar destaque. Se quiser deixar o texto em itálico deve-se usar a tag <em>, se quiser deixar uma parte do texto marcado utiliza-se a tag <mark>ou ainda a tag <small> que torna uma palavra ou texto menor.
Outros exemplos de tag são: se desejar que o texto seja desconsiderado utilizamos a tag <del>. Já se quiser dar destaque para uma frase com sublinhado em baixo do texto utilizar<ins>.

Também é possível adicionar textos sobrescritos e subscritos para o caso de fórmulas matemáticas, para que o caracter fique na parte superior deve utilizar a tag <sup> como por exemplo x2+1 e para que o caracter fique na parte inferior deve utilizar a tag<sub> como por exemplo: H2O ou log510.

Listas

Listas Ordenadas

São listas em que a ordem dos itens é de extrema importância como a classificação de uma prova para isso é necessário utilizar <ol> para delimitar a lista e <li> listar os itens,já para identificar/numerar cada item da lista se será números ou letras na tag <ol> deve-se definir o type

  1. Vanessa
  2. Amanda
  3. Thiago

Listas não Ordenadas

São listas em que a ordem não importa, como uma lista de supermercado, utiliza-se <ul> e também é possível mudar seu type para por exemplo square, disc,circle e também deve-se usar <li> para listar oos itens:

  • arroz
  • feijão
  • massa

Lista mista

Lista em que a listas ordenadas e não ordenadas estão misturas, exemplo liguangens de programação

  1. Já estudadas
    • c
    • java
  2. Em estudo
    1. pyhton
    2. c#

Lista de definição

Lista que trás uma palavra e sua definição. Utiliza-se a tag <dl> para iniciar a lista e as tags <dt> onde vai a palavra e <dd> onde vai a definição da palavra
