Semâtinca

Para deixar uma ou mais palavras em negrito é preciso usar a tag <strong> entre as palavras que se quer dar destaque. Se quiser deixar o texto em itálico deve-se usar a tag <em>, se quiser deixar uma parte do texto marcado utiliza-se a tag <mark>ou ainda a tag <small> que torna uma palavra ou texto menor. Outros exemplos de tag são: se desejar que o texto seja desconsiderado ou silanalizar que ele deve ser deletado utilizamos a tag <del>. Já se quiser dar destaque para uma frase com sublinhado em baixo do texto utilizar<ins>.

Também é possível adicionar textos sobrescritos e subscritos para o caso de fórmulas químicas ou matemáticas, para que o caracter ou caracteres fique na parte superior(sobrescrito) deve utilizar a tag <sup> como por exemplo x2+1 e para que o caracter fique na parte inferior (subscrito) deve utilizar a tag<sub> como por exemplo H2O ou log510.

HTML é a linguagem padrão utilizada para criar páginas da Web. Ela fornece a estrutura básica do conteúdo, permitindo que textos, imagens, links, formulários e outros elementos sejam organizados de forma hierárquica. O HTML é linguagem de marcação, cujo objetivo é descrever o significado e a estrutura das informações apresentadas ao usuário. Por meio de suas tags, o HTML indica ao navegador como cada parte do conteúdo deve ser interpretada e exibida.(DUCKETT, 2014, p. 22)
