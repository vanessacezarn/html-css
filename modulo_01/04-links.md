# LINK
* hyperlinks permitem ir de um ponto a outroo na World Wide Web, ou seja, quando em um site e clica em um local para ir para outra página, site ou baixar um arquivo
* para criar um hiperlink é preciso criar âncoras através da tag < a> em que o principal atributo é href que cria uma referência de hipertexto 
    * para links que iram direcionar para páginas fora do site utilizar target="_blank" rel="external" logo após o href para que o link seja aberto em uma nova janela do navegador. 
    * Exemplo: Acesse< a href="https://github.com/vanessacezarn" target="_blank" rel="external"> meu repositórios do GitHub< /a> para ver meus projetos 
* para links que direcionam para páginas do meu própio site não precisa colocar o https... basta apenas colocar o caminha do .html  no href, mas cuidado se o .html estiver dentro de outra pasta é necessario colocar todo caminho até ele. 
* para realizar download de arquivos de dentro do meu site  utiliza-se o mesmo principio do redirecionamento mas dentro da servidor do site tem que estar disponivél o arquivo que será disponibilizado para dowload e caminho para ele tem que estar completo dentro do href
