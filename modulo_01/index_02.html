Imagens Flexíveis

São imagens que adaptam ao tamanho da tela do navegador, agora além da tag <img> tem temos que usar no inicio a tag <picutre> e a tag <source>.
Dentro de <source> temos 3 atributos type que indica o media type da image,srcset que configura o nome da imagem que será carregada quando o tamanho for atingindo e media indica o tamanho máximo a ser considerado para carregar a imagem indicada em srcset
Para testar temos 3 imagens uma de cada tamanho, então dependo do tamanho que estiver o navegador vai abrir uma imagem diferente.

Vídeos

Se o vídeo estiver hospedado no própio servidor. Utilizar a tag <video> .
Alguns atributos da tag <video>:

  • width = indica a largura que o vídeo ocupará na tela
  • poster = capa do vídeo, enquanto o vídeo não é reproduzido
  • controls = configura se os controles do vídeo vão aparecer na parte inferior, para ativar os controles adicionar controls na tag video
  • autoplay = diz para o navegador tocar o vídeo assim que a página for carregada
O correto é ter os vários formatos de vídeos para que ele possa ser reproduzido nos diferentes navegadores, dai fazer que nem nos audios.
