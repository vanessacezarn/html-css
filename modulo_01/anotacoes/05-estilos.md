# ESTILOS CSS
### CSS inline style
* utilizando CSS dentro de parâmetros de HTML5
* para mudar a cor de fundo da página na tag < body> adicionar style:background-color:
* para centralizar um título dentro da tag < h1> adicionar o sytle:text-align
* se quiser mudar a cor das letras dentro da tag de titulo ou paragrafo adicionar style:color
* se fizer utilizando esse método tem ajustar tag por tag
### CSS internal style
* tag < style> dentro da área < head> e para cada tag que deseja mudar o estilo colocar o nome da tag segiudo de chaves e dentro das chaves adicionar as aleterações que deseja fazer. 
### CSS external style
* utilizado para não precisar ficar copiado o css internal style para cada página 
* utilizando a tag < link>, configurada para trabalhar com arquivos externos de estilos, dentro do < head>
* criar um arquivo style.css e nele colocar as configurações de estilo que deseja para as páginas, dessa forma quando for precisar fazer uma alteração basta somente mexer nesse arquivo.
* se precisar fazer uma configuração pontual em algum lugar pode-se utilizar o css internal ou inline style.

#### dar preferência para external style
