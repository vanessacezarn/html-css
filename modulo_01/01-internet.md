### DARPA - surgiu na guerra fria
  * ARPANET - comunicar centros militares, formou uma rede por todo os EUA
  * NCP - Network Control Protocol, permitia a comunicação apenas entre 2 computadores por vezes, para toda a rede para que ele pudessem se comunicar
  *  TCP/IP - Transmission Control Protocol/Internet Protocol, divide o pacote a ser transmitido em pacotes menores para fazer a transmissão
### WWW - Word Wide Web, conjunto de servidores especializados em http 
### representação de dados --> onda quadrada 
  * bit --> 0 ou 1
  * 8 bits = 1 byte
  * 1024 bytes = 1 Kilo Bytes(KB)
  * 1024 KB = 1 Mega Bytes(MB)	
* MB (mega bytes -> armazenamento) != Mb (Mega bits -> transmissão)
### modem --> responsavel por modular e demodular os sinais que saem e chegam
### DNS - (Domain Name System) - "agenda telefônica", traduz nomes de sites para endereços IP
### domínio
  * nome que identifica o site
  * é único
  * pago anualmente
  *  TLD (Top-Level Domain) - extensão final de um endereço de internet, situada após o último ponto, indica a finalidade, o tipo de organização ou a localização geográfica de um site
### hospedagem
  * local onde o site está armazenado
  * espaço para armazenar arquivos
### URL (Uniform Resource Locator) - endereço único de um recurso na internet
  * dividida em **sub-dominio**, ==domínio==, **TLD**, ==caminho==
  * exemplo: **www** . ==github== **com**/==gustavoguanabara==
